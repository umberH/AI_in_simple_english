# AI in Simple English

Learn LLMs from first principles through code you build yourself.

41 technical posts covering tokenization → embeddings → transformers → RAG → agents → production deployment.

## What's Inside

**Foundations** — Tokens · BPE · Embeddings · Attention · Transformers
**Build** — Pre-training · Sampling · Alignment · Quantization · Local inference
**Measure** — RAG · Retrieval evaluation · LLM-as-judge · CI/CD for evals
**Orchestrate** — Function calling · ReAct · Memory · Multi-agent systems
**Ship** — Caching · Observability · Cost monitoring · Docker · Terraform

Each post includes full code, diagrams, and hands-on examples.

## Getting Started

```bash
git clone https://github.com/umberH/ai-in-simple-english
cd ai-in-simple-english
python posts/01-tokens/code/tokens.py
```

Start with [posts/01-tokens](posts/01-tokens/draft.md) to see how tokenization works.
